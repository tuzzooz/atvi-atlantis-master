## Pré-requisitos

Antes de começar, você vai precisar ter as seguintes ferramentas instaladas em sua máquina:
* **Node.js** (v22.13.1 ou similar)
* **TypeScript**

---

## Como Rodar o Projeto

Siga estes passos para configurar e executar o teste principal:

### 1. Instalar Dependências

Este projeto pode ter dependências (como o `prompt-sync` para entrada de dados). Abra o terminal na pasta raiz do projeto e instale o que for necessário:

npm install

npm install -g typescript

tsc

node js/teste/index.js
